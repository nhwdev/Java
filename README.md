## 📖 커리큘럼

| 단계 | 폴더 | 주제 | 핵심 개념 |
|:----:|------|------|-----------|
| 1 | `chap01_helloworld` | HelloWorld | JVM 구조, 기본 출력 |

| 3 | `chap03_operator` | 연산자 | 산술/비교/논리/삼항 연산자 |

| 5 | `chap05_array` | 배열 | 1차원/2차원 배열, Arrays |
| 6 | `chap06_class` | 클래스 | 객체, 생성자, 접근제한자 |
| 7 | `chap07_inheritance` | 상속 | extends, super, 오버라이딩 |
| 8 | `chap08_interface` | 인터페이스 | implements, 다형성 |
| 9 | `chap09_inner` | 내부 클래스 | 중첩/익명/람다 클래스 |
| 10 | `chap10_exception` | 예외 처리 | try-catch, throws, 커스텀 예외 |
| 11 | `chap11_api` | 기본 API | String, Math, Date, Object |
| 12 | `chap12_collection` | 컬렉션 프레임워크 | List, Set, Map, Iterator |
| 13 | `chap13_iostream` | 입출력 스트림 | InputStream, OutputStream, Reader/Writer |
| 14 | `chap14_thread` | 멀티 스레드 | Thread, Runnable, 동기화 |
| 15 | `chap15_stream` | 스트림 | Stream API, filter, map, reduce |
| 16 | `chap16_jdbc` | JDBC | Connection, PreparedStatement, CRUD |
| 🏁 | `coding_test` | 코딩테스트 | BOJ 문제 풀이 |
